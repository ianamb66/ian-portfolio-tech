# Ian Aceves Portfolio

Portfolio profesional de Ian Miguel Aceves Mejía Bringas.

## Stack

- React
- Vite
- GSAP
- Lucide React

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Enfoque

Sitio one-page con dirección visual tecnológica/editorial inspirada en sistemas de interfaz, retícula visible, motion sutil y secciones orientadas a vender perfil profesional, casos, herramientas y contacto.
